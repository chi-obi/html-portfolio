# html-portfolio
The focus was to create my first Web Developer Portfolio website using HTML. This portfolio serves as a platform to showcase my work and allow potential employers or clients to view my projects and learn more about me.

Key Tools and Concepts Learned:
1. Basic HTML structure and elements, including titles, subtitles, and anchor tags.
2. How to take screenshots and manage images to include in the portfolio.
3. Use of Visual Studio Code (VS Code) as my Integrated Development Environment (IDE) to edit HTML files.
4. Best practices for organizing files (such as HTML and image files) within my project directory.
   
Project Goals:
The goal was to create a simple HTML-only portfolio website, incorporating my actual work and personal touches. This includes adding links to previous projects.

Learning Outcomes:
1. Developed a portfolio that not only showcases my work but also enhances my visibility in the web development industry.
2. Customized a template and integrated my content effectively.
3. Addressed important aspects of web design, such as image sizing and layout.
4. Published my portfolio online.
5. Have a personal website to promote myself in the industry.
